# Sistemas-de-Bitacoras-Digitales
Sistema de bitácoras desarrollado mediante Microsoft Power Apps y SharePoint con el objetivo de digitalizar y centralizar las bitácoras operativas del área de TI.

La información capturada se almacena de forma centralizada en SharePoint, facilitando las consultas históricas de registros, el seguimiento de actividades y el control de las tareas realizadas dentro del área de TI.

La implementación de esta solución contribuye a mejorar la organización de la información, reducir el uso de papel y mantener un mayor control sobre las actividades registradas por el área.

# Problemática

Antes de esta implementación, las bitácoras se realizaban en formato físico, lo que hacía más lento el proceso de registro, almacenamiento y consulta de la información.

La gestión de las bitácoras dependía de documentos impresos, por lo que la búsqueda del registro histórico requería revisar manualmente cada formato y bitácora, dificultando la localización de información y el seguimiento de actividades realizadas por el área de TI.

Además, el almacenamiento físico de las bitácoras incrementaba el riesgo de pérdida, deterioro o extravío de la información, limitando la disponibilidad de los registros y afectando el control de las actividades operativas.

Ante esta necesidad, se desarrolló una solución digital basada en Microsoft Power Apps y SharePoint con el objetivo de centralizar la información, facilitar la consulta de registros históricos y mejorar el control de las actividades documentadas por el área de TI.


# Objetivo
Desarrollar una solución digital para la gestión, administración y creación de bitácoras operativas y de mantenimiento preventivo y correctivo, permitiendo centralizar la información, facilitar la consulta de registros históricos y mejorar el control de las actividades realizadas mediante el uso de Microsoft Power Apps y SharePoint.

# Tecnologías Utilizadas

* Microsoft Power Apps
* Microsoft SharePoint
* Microsoft Teams
* Microsoft 365

# Funciones Principales

* Registro de bitácoras operativas.
* Registro de mantenimientos preventivos.
* Registro de mantenimientos correctivos.
* Consulta de registros históricos.
* Almacenamiento centralizado de información.
* Seguimiento de actividades del área de TI.
* Gestión digital de bitácoras.
